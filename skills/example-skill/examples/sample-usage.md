# Example Skill Usage

## Basic Usage

To use this skill, invoke it with:

```
/example-skill
```

Or let Claude automatically invoke it when relevant to the task.

## Example Workflows

### Workflow 1: Validate Configuration

```
Use the example-skill to validate the current configuration
```

Expected output:
- Configuration validation runs
- All checks pass
- Summary of findings displayed

### Workflow 2: Deploy with Validation

```
Use the example-skill to validate and deploy the application
```

Expected output:
- Validation checks run first
- If validation passes, deployment proceeds
- Deployment status reported

## Sample Prompts

1. **Basic validation**: "Validate the project setup using example-skill"
2. **Deployment**: "Deploy the application using the example-skill"
3. **Configuration check**: "Check if the configuration is valid"

## Integration with Other Skills

This skill can be combined with:
- Testing skills for pre-deployment validation
- Monitoring skills for post-deployment checks
- Documentation skills for generating deployment reports
